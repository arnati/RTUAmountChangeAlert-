# Оповещение менеджера об изменении суммы в РТУ (УТ 11.5)

---

## Описание задачи
**Расширение** для **1С:УТ 11.5**, которое **отслеживает изменения суммы в документе "Реализация товаров и услуг"** и **автоматически создает задачу основному менеджеру клиента** для согласования изменений. Задача создается с темой **"Требуется согласование изменения суммы документа РТУ"** и содержит ссылку на документ реализации, а также информацию о предыдущей и новой сумме.

---

## Исходные данные
- **Платформа**: 1С:Предприятие 8.3.27.
- **Редакция УТ**: 11.5.11.70.
- **Тип доработки**: Расширение конфигурации.
- **Задача**: Бизнес-процесс "Задание" (создается через программный код).

---

## Возможности

### 1. Отслеживание изменений
- Сравнение текущей суммы документа с предыдущей.

### 2. Создание задачи
- Задача создается для основного менеджера клиента.
- Тема задачи: **"Требуется согласование изменения суммы документа РТУ"**.
- В описании задачи указывается ссылка на документ реализации, а также предыдущая и новая сумма.
- Срок исполнения задачи: 7 дней с момента создания.

---

## Особенности реализации
- **Переопределение обработчиков**: Используется для отслеживания изменений в документе.
- **Создание задач**: Используется бизнес-процесс "Задание".

## Как использовать
1. **Установка расширения**:
   - Подключите расширение к конфигурации **1С:Управление торговлей 11.5**.

2. **Оповещение**:
   - При изменении суммы в документе "Реализация товаров и услуг" основному менеджеру клиента будет создана задача на согласование.

---

## Результаты
- Автоматизация процесса согласования изменений в документах реализации.
- Снижение риска несанкционированных изменений.
- Уведомление ответственных лиц об изменениях в документах.
